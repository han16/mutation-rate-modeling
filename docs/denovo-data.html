Data discretization and categorization

  • Continuous features are categorized into 50 quantile intervals, and observations in every interval use the same discretized value of the mean
  • use R function Split to cluster windows with similar features including discrete features and categorized features

Model fitting

We use two methods to fit the model

  • GLM (use R function GLM)
  1. The response is the ** total number of mutations ** in the category and the each feature is averaged across all widnows within the category (denoted as M1 (method 1))
  2. The response is the ** averaged number of mutations (mutation rate) ** in the category and the each feature is averaged across all widnows within the category (denoted as M2 (method 2))
  3. intercept is included and the same across all categories
  4. confidence interval could be obtained as well
  • use R function optim to fit the model (M3 method 3)
  1. use estimated baseline mutation rate in the model.
  2. There is a singularity problem for hessian matrix when getting the confidence interval by optim.
